	這一題有個關鍵點，它要的是第一頭牛和第N頭牛之間
的「最大」距離。假設有兩頭牛，他們之間的距離不能超
過10，按照算法導論中的建圖方式，會長這樣：
               v0 ----0---->v1
			    |           |
				|		    |
				|		    |
				|		   10
				0		    |
				|		    |
				|		    |
				|			V
				----------->v2
最後的結果，第一頭牛和第二頭牛會被放在同一個位置，
這當然滿足距離不能超過10的要求，但答案應該是10，因
為題目要求距離最遠。因此我們會想讓最短路通過那條10
的路，唯一的方法，就是把v0移除，然後以v1為起點，計
算v1到vn的最短距離。
